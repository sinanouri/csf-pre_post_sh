# ConfigServer Security & Firewall (CSF) - csfpre.sh / csfpost.sh

## Install
Installation is quite straightforward:

```
# cd /usr/local/src
# git clone https://github.com/sinanouri/csf-pre_post_sh.git
# cd csf-pre_post_sh
# sh install.sh
# cd csf-pre_post_sh
# sh install.sh
```
